### Alternatives
- [build123d](https://github.com/gumyr/build123d?tab=readme-ov-file#----) - a Python-based, parametric boundary representation (BREP) modeling framework for 2D and 3D CAD. Built on the Open Cascade geometric kernel, it provides a clean, fully Pythonic interface for creating precise models suitable for 3D printing, CNC machining, laser cutting, and other manufacturing processes.
- [CadQuery](https://github.com/CadQuery/cadquery?tab=readme-ov-file#cadquery) - an intuitive, easy-to-use Python module for building parametric 3D CAD models. CadQuery's CAD kernel Open CASCADE Technology (OCCT) is much more powerful than the CGAL used by OpenSCAD. Features supported natively by OCCT include NURBS, splines, surface sewing, STL repair, STEP import/export, and other complex operations, in addition to the standard CSG operations supported by CGAL.
- [sdf](https://github.com/fogleman/sdf?tab=readme-ov-file#sdf) / [HN](https://news.ycombinator.com/item?id=46339316) - Generate 3D meshes based on SDFs (signed distance functions) with a dirt simple Python API.
- [DSLCAD (Pronounced DieselCAD)](https://github.com/DSchroer/dslcad) / [HN](https://news.ycombinator.com/item?id=34740463)
- [CadQuery](https://github.com/CadQuery/cadquery) / [HN](https://news.ycombinator.com/item?id=30232344)
