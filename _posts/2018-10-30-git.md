## Basic Cmd

### Undo last commit, keeping change

{% highlight bash %}
git reset HEAD^
{% endhighlight %}

### Abort merge

{% highlight bash %}
git reset --hard HEAD
git clean -fd  
{% endhighlight %}

### Checkout branch

{% highlight bash %}
# without using origin/xx
git checkout <branch from above>    
{% endhighlight %}
