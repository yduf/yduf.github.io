### Emblems
- [Backup emblems](https://forums.linuxmint.com/viewtopic.php?t=281724) - ` emblems are stored in ~/.local/share/gvfs-metadata`

{% highlight bash %}
$ gio info /path/to/folder | grep emblems # get embles
$ gio set # set emblems
{% endhighlight %}
