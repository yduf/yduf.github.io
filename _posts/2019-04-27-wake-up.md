## Power Buton
- [/etc/systemd/logind.conf](/etc/systemd/logind.conf)
{% highlight cpp %}
HandlePowerKey=poweroff
{% endhighlight %}
sudo service systemd-logind restart

## Wake-On-Lan
{% highlight cpp %}
wakeonlan MAC Address
{% endhighlight %}

Get MAC Address with `cat /sys/class/net/ens1f0/address ` or `ip link show`
