## GH CLI Tools

Theres is a gh CLI to interact directly with github from a remote term (once authenticatd)

{% highlight bash %}
$  sudo apt-get install gh

$ gh auth login

$ gh repo list yduf --visibility public
$ gh repo list yduf --visibility private
{% endhighlight %}
