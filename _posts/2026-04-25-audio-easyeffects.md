# [Voice Filter](https://chatgpt.com/share/69ece72f-1c7c-83eb-b570-cca3059d57c3)

Ajoute ces filtres (dans cet ordre) :

1. Noise Gate (porte de bruit)
Close threshold : ~ -45 dB
Open threshold : ~ -35 dB

👉 Coupe les bruits ambiants quand tu ne parles pas

2. Compressor
Ratio : 3:1 ou 4:1
Threshold : ~ -18 dB
Attack : 5–10 ms
Release : 100 ms

👉 Rend ta voix plus homogène

3. Limiter
Threshold : -1 dB

👉 Évite toute saturation finale

4. Amélioration du son 
EQ :
Coupe bas (80 Hz) → enlève les vibrations
Boost léger 3–5 kHz → clarté voix

De-esser (si “s” trop agressifs)

Compresseur plus fin que celui d’OBS
