> [etckeeper](https://chatgpt.com/share/69cfa1c5-0894-8328-80c8-1d25cdf5e33b) is a handy tool that lets you track changes in your /etc directory (where most Linux system configuration lives) using a version control system like Git.

- [SaltStack](https://saltproject.io/) - an event-driven automation tool and framework to deploy, configure, and manage complex IT systems. 
  - Managing operating system deployment and configuration.
  - Installing and configuring software applications and services.
  - Ensuring consistent configuration and preventing configuration drift.

# Usage
Etckeeper automatically integrates with your package manager:
- On apt (Debian/Ubuntu): commits before/after package installs

**see also**
- [How to keep configuration files under version control with Etckeeper](https://linuxconfig.org/how-to-keep-configuration-files-under-version-control-with-etckeeper)

{% highlight bash %}
$ sudo apt install etckeeper
{% endhighlight %}

{% highlight bash %}
$ sudo etckeeper init
$ sudo etckeeper commit "Initial commit"
{% endhighlight %}

## Viewing changes

{% highlight bash %}
$ cd /etc
$ sudo git log
$ sudo git diff

# or
$ sudo etckeeper vcs log
$ sudo etckeeper vcs diff
{% endhighlight %}
